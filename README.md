# DiscordPollBot-front

## DiscordPollBotのフロント側を作ります

Discord上で実施された投票結果を可視化・閲覧するためのフロントエンドアプリケーションです。  
本プロジェクトは、DiscordPollBot（バックエンド）と連携し、投票結果の一覧表示や詳細確認を行うことを目的としています。

---

## 概要

Discord Poll Bot によって収集・保存された投票データを、  
WebUIとして分かりやすく表示するフロントエンドです。

- 投票結果の一覧表示
- 各投票の詳細確認
- 技術構成・設計方針・参考文献をまとめた Docs ページの提供

なんちゃって保守性を意識した設計を行っています。
